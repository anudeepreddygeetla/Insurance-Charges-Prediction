# Insurance Charges Prediction

## 📌 Project Overview

This project predicts medical insurance charges using Machine Learning.
The model analyzes factors like age, BMI, smoking habits, and other features to estimate insurance costs.

---

## 📊 Workflow

* Data Cleaning
* Exploratory Data Analysis (EDA)
* Data Preprocessing
* Model Training
* Model Evaluation

---

## 🤖 Model Used

* Linear Regression

---

## 📈 Results

* The model was able to predict insurance charges with good accuracy
* Linear Regression performed well for this dataset

---

##  Features Used

* Age
* Sex
* BMI
* Children
* Smoker
* Region
